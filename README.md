## 깃 커밋 컨벤션
* `feat`: 새로운 기능 개발
* `fix`: 버그 수정
* `add`: 새로운 기능 추가
* `chore`: 빌드 업무 수정 (build.gradle 수정)
* `refactor`: 코드 리팩토링
* `style`: 코드 포맷팅, 주석 삭제 등 코드 변경이 없는 경우
* `test`: 테스트 코드, 리팩토링 테스트 코드 개발 및 추가
